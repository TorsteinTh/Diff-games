Mayham game






Please enter    $python3 main.py       to start the game
                ----------------
--------------------------------------------------------



Player one        |      Player Two      |          Action
---------------------------------------------------------
UP    - arrow     |        'w'           |          Engine
LEFT  - arrow     |        'a'           |          Rotate left  
RIGHT - arrow     |        'd'           |          Rotate Right
DOWN  - arrow     |        'spacebar'    |          Fire shots



Others
---------------------------------------------------------
'r'                                      Restart
Esc                                      Exit

Tip: Don't die
